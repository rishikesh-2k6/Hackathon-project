# Hackathon Project - Interactive Tutorial System

## Overview
This interactive plug-in replaces video tutorials with real-time on-screen guidance. Using dynamic pointers to show exactly where to click, it helps users master software quickly without leaving the application window.

## Features
- Real-time on-screen guidance
- Dynamic pointers for precise navigation
- No need to switch between tutorial videos and applications
- Interactive learning experience

## Getting Started
1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the main application: `python pro-3-main2.py`
